Mô tả Công việc
- Lập trình ứng dụng xử lý hình ảnh, machine learning và deep learning trên các hệ điều hành như Linux, Windows hoặc MacOS.

- Xây dựng ML model để giải quyết các bài toán Classification, Object detection, Semantic segmentation, Instance segmentation, OCR...

- Chuẩn bị dataset từ các dữ liệu thu thập trên các hệ thống trong các dự án của công ty.

- Nghiên cứu ứng dụng các giải pháp machine learning và deep learning vào các dự án hiện tại và các dự án mới.

- Xây dựng process software, xử lý hình ảnh, quản lý tài nguyên camera, quản lý tài nguyên GPU.

- Tối ưu ML model và xây dựng project có kiến trúc rõ ràng.
